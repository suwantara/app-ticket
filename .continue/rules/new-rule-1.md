# Documentation Resources

For framework-specific questions, refer to:

- Tailwind CSS: https://tailwindcss.com/docs
- Alpine.js: https://alpinejs.dev/start-here
- Laravel: https://laravel.com/docs
- Livewire: https://livewire.laravel.com/docs
- Flowbite: https://flowbite.com/docs/getting-started/introduction/

Always cite documentation when explaining concepts.

# Coding Standards

- Use Livewire components for interactive features
- Prefer Alpine.js for client-side interactivity
- Follow Laravel conventions for models, controllers, and routes
- Use Flowbite components with Tailwind utility classes
- Keep Blade templates clean and component-based
